# Kauvio Complete AI Build V3

Dieses Paket kombiniert:
- den grossen Backend-/AI-Block
- den finalen Frontend-/Admin-Branding-Block
- Homepage nur 6 Vergleiche
- Suchergebnis mit ähnlichen Produkten + Vorschlägen
- AI Discovery + Extract + Match + Compare

## Enthalten
- services/ai_service/ai_service.mjs
- services/worker/open_web_discovery.mjs
- webapp/server/index.mjs
- webapp/server/canonical_search_runtime.mjs
- webapp/src/App.jsx
- webapp/src/Root.jsx
- webapp/src/HomePageProfessional.jsx
- webapp/src/SearchSuggestBox.jsx
- webapp/src/Brand.jsx
- webapp/src/styles.css
- webapp/src/swiss-home.css
- webapp/package.json

## Zielbild
- KI findet Produkte automatisch
- KI liest Produktseiten aus
- gleiche Produkte werden gemerged
- Vergleiche werden aufgebaut
- Startseite zeigt nur 6 Karten
- Suchergebnis zeigt ähnliche Produkte und weitere Vorschläge
- Admin ist funktional für Starten, Anzeigen und Steuern
